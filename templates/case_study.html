The Crypto Swap Reseller Problem

How "zero-fee" swap services silently charge you 1-5% through hidden rate markup — and how to verify it yourself.

The Pattern

NEAR Intents (formerly Defuse/Chain Defuser) offers a swap API called 1Click. It aggregates liquidity from decentralized market makers and provides cross-chain swaps. The protocol charges zero fees. The API is publicly documented.

A growing number of swap services have built frontends on this API. They present themselves as independent exchanges, but they're resellers. They take the NEAR Intents rate, add a silent markup of 1-5%, and pocket the difference. The user never sees the original rate.

These services don't disclose that they're using NEAR Intents. They don't show the pre-markup rate. The markup isn't listed as a "fee" — it's baked into the exchange rate itself, making it invisible unless you compare against the source.

How to Identify a Reseller

Every NEAR Intents reseller shares these telltale signs:

  1. Deposit addresses on NEAR: They give you a deposit address that's a NEAR account or uses NEAR-based smart contracts. The deposits go through Defuse/NEAR infrastructure regardless of what chain you're swapping.
  2. Identical supported token pairs: Their token list matches exactly what NEAR Intents supports — no more, no less. When NEAR Intents adds a new chain, they add it too.
  3. Correlation IDs: The API returns correlation IDs that show up in NEAR Intents' status endpoints. You can check any deposit address against the public NEAR Intents status API.
  4. Rate discrepancies: Request the same swap on the reseller and directly through NEAR Intents. The reseller's rate will be worse by exactly their markup percentage.

How to Verify Yourself

You don't need to trust us — or anyone. Here's how to verify any swap service:

  1. Request a quote from the swap service for a specific pair and amount.
  2. Request the same quote directly from the NEAR Intents API. The endpoint is POST https://1click.chaindefuser.com/v0/quote with dry: true.
  3. Compare the exchange rates. If the swap service gives you less than NEAR Intents, the difference is their hidden markup.
# Get a dry quote directly from NEAR Intents
curl -X POST https://1click.chaindefuser.com/v0/quote \
  -H "Content-Type: application/json" \
  -d '{
    "dry": true,
    "swapType": "EXACT_INPUT",
    "slippageTolerance": 100,
    "originAsset": "nep141:eth.near",
    "depositType": "ORIGIN_CHAIN",
    "destinationAsset": "nep141:usdt.near",
    "amount": "1000000000000000000",
    "refundTo": "YOUR_ADDRESS",
    "refundType": "ORIGIN_CHAIN",
    "recipient": "YOUR_ADDRESS",
    "recipientType": "DESTINATION_CHAIN",
    "deadline": "2025-12-31T00:00:00Z"
  }'

Replace addresses with your own. The amount is 1 ETH in wei (18 decimals).
